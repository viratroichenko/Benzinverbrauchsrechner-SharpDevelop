
# Benzinverbrauchsrechner (C# Console Application)

## Beschreibung

Dieses Projekt berechnet den Benzinverbrauch eines Autos auf 100 km.

Der Benutzer gibt den Kilometerstand vor dem letzten Tanken, den aktuellen Kilometerstand, die getankte Menge an Kraftstoff sowie seinen Namen ein. Das Programm berechnet anschließend die gefahrene Strecke und den durchschnittlichen Verbrauch.

## Funktionen

Eingabe von:

* Kilometerstand vor dem letzten Tanken
* Aktueller Kilometerstand
* Getankte Liter
* Name des Fahrers / der Fahrerin

Berechnung von:

* Gefahrene Kilometer seit dem letzten Tanken
* Benzinverbrauch auf 100 km

Ausgabe von:

* Gefahrene Strecke
* Verbrauchte Liter
* Durchschnittlicher Verbrauch pro 100 km

## Technologien

* C#
* .NET (Console Application)

## Ziel

Ziel des Projekts war es, grundlegende Konzepte wie Benutzereingaben, Variablen, mathematische Berechnungen und einfache Programmlogik zu üben.
